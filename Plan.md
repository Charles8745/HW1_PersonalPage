# HW1 Personal Page

## Task1 Personal Webpage Created

### 1. Built a modern, single-page personal website for Chen Xu-Lin (index.html) 

- **Features:**
    - Name displayed with a purple gradient shimmer effect
    - Live clock updating every second with hours, minutes, and seconds
    - Full date display (e.g., *Tuesday, February 24, 2026*)

- **Design:**
    - Glassmorphism card with frosted-glass effect and backdrop blur
    - Animated gradient background (deep purple/indigo tones)
    - Three floating, blurred orbs for visual depth
    - Smooth staggered entrance animations (fade-up)
    - Hover lift effect on the card
    - Responsive layout for mobile and desktop
    - Google Fonts (Outfit & Inter) for modern typography
